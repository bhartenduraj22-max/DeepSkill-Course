### GOF(Gang of Four)
4 Author who wrote the famous book.
Introduce 23 Design Pattern that help to solve common software design problem.
## GOF pattern are divided into 3 Group
# Creational Pattern   
Involve how to create object
# Structural Pattern
How to Organise object
# Behavioural Pattern
How Object talk to each other
# Architectural Plan
* MVC(Model View Controller)
  Model : Data
  View : Show to Screen
  Controller : Connect model and view
* Dependency Injection
  Use ready Made Object Instead of Creating yourself
